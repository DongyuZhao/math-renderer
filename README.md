# math-renderer

Cross-platform package scaffold for rendering MathJax-compliant LaTeX formulas.

## Packages in this repository

- **npm package** (root `package.json`)
- **Swift Package Manager package** (root `Package.swift`, sources under `swift/`)
- **Android AAR library** (Gradle project under `android/`)

## Publish targets

- npm: `@dongyuzhao/math-renderer`
- SwiftPM: `MathRenderer`
- Android Maven artifact (AAR): `io.github.dongyuzhao:math-renderer-android`
